# Quickstart Guides

Per-app guides to get running fast. Each guide is self-contained.

| Guide                          | What You Get                          |
| ------------------------------ | ------------------------------------- |
| [Monorepo](./monorepo.md)      | Full stack running locally            |
| [Profiles](./profiles.md)      | Choose a deployment profile (Minimal / SaaS / Admin / Advanced) |
| [Web](./web.md)                | Vite + React frontend only            |
| [Server](./server.md)          | Fastify API + PostgreSQL              |
| [Desktop](./desktop.md)        | Electron app wrapping the web client  |

**Recommended starting point:** [Monorepo](./monorepo.md) -- starts everything with one command.

**Choosing a profile:** [Profiles](./profiles.md) -- configure what features are active via environment variables.

For deployment guides, see [`apps/docs/deploy/`](../deploy/README.md).
