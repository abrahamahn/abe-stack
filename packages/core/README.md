# @abe-stack/core

This package contains the core functionality for abe-stack. It includes API contracts, shared types, and validation schemas.

## Installation

```bash
pnpm add @abe-stack/core
```

## Usage

This package provides a set of utilities and types that are used across the abe-stack monorepo.

### Contracts

API contracts are defined using `zod` and `@ts-rest/core`.

### Types

Shared types are located in the `src` directory.

### Validation

Validation schemas are built using `zod`.
