# Commands

## Development

- `pnpm dev`
- `pnpm dev:web`
- `pnpm dev:server`

## Quality

- `pnpm format`
- `pnpm lint`
- `pnpm lint:fix`
- `pnpm type-check`
- `pnpm test`

## Build

- `pnpm build`
- `pnpm build:web`
- `pnpm build:server`

## Database

- `pnpm db:generate`
- `pnpm db:migrate`
- `pnpm db:seed`
